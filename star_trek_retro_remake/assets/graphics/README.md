# Star Trek Retro Remake - Graphics Assets

This directory contains all graphical assets for the game.

## Directory Structure

- `ships/` - Starship sprites and animations
- `ui/` - User interface elements and icons
- `effects/` - Visual effects and particles
- `backgrounds/` - Background images and textures
- `icons/` - Game icons and symbols

## Asset Requirements

- All graphics should be in PNG format with transparency support
- Sprites should be designed for pixel-perfect rendering
- Use consistent color palette for UI elements
- Maintain 16:9 aspect ratio for background images
