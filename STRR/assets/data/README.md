# Star Trek Retro Remake - Data Assets

This directory contains game data files and configurations.

## Directory Structure

- `sectors/` - Sector map definitions and layouts
- `missions/` - Mission scripts and objectives
- `dialogs/` - Dialog text and conversation trees
- `saves/` - Saved game files (user generated)

## Data Format

All data files use TOML format for easy editing and version control.
Files should maintain proper TOML syntax and structure.

## Content Guidelines

- Keep file sizes reasonable for performance
- Use descriptive naming conventions
- Maintain consistency with game data schema
- Test all data files for valid TOML syntax
